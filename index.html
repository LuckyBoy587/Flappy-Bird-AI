üìã State Space

The AI observes 5 normalized state variables:

  • bird_y: Vertical position of the bird (0-1)
  • bird_velocity: Vertical velocity (-1 to 1)
  • next_pipe_x: Horizontal distance to next pipe (0-1)
  • next_pipe_top: Top of the pipe gap (0-1)
  • next_pipe_bottom: Bottom of the pipe gap (0-1)

üéØ Reward System

  • +1.0: Successfully passed through a pipe
  • +0.1: Survived another frame
  • -100: Collision with pipe, ground, or ceiling

üõ†Ô∏è Technical Details

Architecture

  • Neural Network: 3-layer fully connected DQN
  • Input Layer: 5 state variables
  • Hidden Layers: 64 and 32 neurons with ReLU activation
  • Output Layer: 2 Q-values (one per action)

Training Algorithm

  • Algorithm: Deep Q-Network (DQN)
  • Experience Replay: 10,000 transitions
  • Target Network: Updated every 100 steps
  • Exploration: Œµ-greedy with decay

Dependencies

  • pygame: Game rendering and physics
  • numpy: Numerical operations
  • torch: Deep learning framework
